# Pet Images

Place pet photos in this directory using the following filenames:

## Jack-Jack
- `jack-jack.jpg` – main profile photo
- `jack-jack-2.jpg` – gallery photo
- `jack-jack-3.jpg` – gallery photo

## Nagi
- `nagi.jpg` – main profile photo
- `nagi-2.jpg` – gallery photo
- `nagi-3.jpg` – gallery photo

Recommended size: 800×800 px, JPEG format.
